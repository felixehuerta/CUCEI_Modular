Aplicación Móvil para la Visualización y Lectura de QR de Árboles – CUCEI

DESCRIPCIÓN DEL PROYECTO
Este proyecto consiste en el desarrollo de una aplicación móvil orientada a la lectura y visualización de códigos QR colocados en los árboles del Centro Universitario
de Ciencias Exactas e Ingeniería (CUCEI) de la Universidad de Guadalajara. Cada código QR permite al usuario acceder a información relevante del árbol, así como a una 
imagen artística creada por estudiantes y vinculada directamente al QR correspondiente. El proyecto integra tecnología, educación ambiental y expresión artística dentro 
del entorno universitario.

OBJETIVO GENERAL
Desarrollar una aplicación móvil interactiva que permita a la comunidad universitaria y visitantes escanear códigos QR instalados en los árboles del CUCEI, consultar 
información detallada de cada árbol y visualizar una obra artística digital asociada.

OBJETIVOS ESPECÍFICOS
- Facilitar el acceso a información botánica mediante el uso de dispositivos móviles.
- Fomentar la conciencia ambiental dentro del campus universitario.
- Integrar el arte estudiantil como parte del proyecto tecnológico.
- Promover el uso de tecnologías móviles y códigos QR con fines educativos y culturales.

FUNCIONALIDADES PRINCIPALES
- Escaneo de códigos QR mediante la cámara del dispositivo.
- Visualización de información del árbol

DESCRIPCIÓN GENERAL
Ubicación dentro del CUCEI
Visualización de una imagen artística creada por alumnos y asociada al árbol.
Interfaz intuitiva y fácil de usar.

ALCANCE DEL PROYECTO
La aplicación estará diseñada para su uso dentro del campus CUCEI y podrá ser utilizada por estudiantes, docentes, visitantes y público en general. 
El proyecto podrá ampliarse para incluir más árboles, nuevas obras artísticas o información adicional.

TECNOLOGÍAS A UTILIZAR
- Plataforma móvil: Android / iOS
- Lenguaje o framework: Flutter / React Native / Kotlin / Swift
- Librerías para lectura de códigos QR
- Base de datos local o servicio en la nube para almacenamiento de información
- Repositorio o servidor para gestión de imágenes artísticas
